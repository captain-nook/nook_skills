# 高密度 AI 媒体标题模式

Use this reference for high-density AI-media titles built around concrete entities, numbers, actions, conflicts, and consequences.

## Observed Surface Traits

The style is built from high-density factual hooks plus dramatic pacing:

- Median-ish length often lands around 30-40 Chinese display characters.
- `!` is common for a first impact beat.
- `,` is common for the second beat: result, contrast, or affected group.
- Numbers are frequent: model version, price, token count, time, benchmark, cost, people count, GPU count.
- English names are frequent: OpenAI, GPT, Claude, Gemini, DeepMind, Meta, Google, H100, API, PR.
- Titles often start with an entity, a number, or a time cue such as `刚刚`.
- The second half usually gives consequence: benchmark result, industry impact, people affected, cost change, or competitive comparison.

## Core Frames

Frame A: entity + explosive action + result

```text
{模型/公司}{强动作}!{数字/结果},{后果/人群影响}
```

Example shape:

```text
Claude再次登顶!化学专业一骑绝尘,全面碾压GPT-4
```

Frame B: release/leak + concrete number + consequence

```text
{产品/模型}{发布/泄露/爆更}!{数字事实},{能力/价格/行业影响}
```

Frame C: cost or efficiency contrast

```text
{实体}{成本/速度变化}!{旧状态/对手},{新结果/人群影响}
```

Frame D: benchmark/ranking conflict

```text
{刚刚/最新},{实体}在{榜单/测试}上{超过/暴锤/登顶}{对手}!{后果}
```

Frame E: human impact

```text
{实体}{能力突破}!{时间/成本/规模},{职业/开发者/研究员}{影响}
```

Frame F: skeptical suspense

```text
{实体}{异常/争议/被查/被曝},{关键问题/后果}
```

Use this when the fact is legal, policy, safety, or controversy related. Keep it steadier; do not overdramatize.

## Common Hook Vocabulary

Use only when factually supported.

- action: 发布, 上线, 爆更, 泄露, 登顶, 碾压, 反超, 开源, 接管, 修复, 降价, 被曝, 遭查, 叫停
- impact: 狂飙, 暴降, 打骨折, 刷榜, 改写, 解放双手, 开始替代, 成本打到新低
- people: 程序员, 开发者, 研究员, 创业者, 学生, 打工人, 企业
- boundary: 传言, 内测, 论文称, 官方称, 社区测试, 网友实测

## Title Construction Rules

- Put the named entity before the abstract topic.
- Prefer one decisive number over several weak numbers.
- If using `刚刚`, the event must be genuinely immediate in the source context.
- If using `碾压`, `吊打`, `暴锤`, or `登顶`, the source must contain a direct comparison or benchmark.
- If using `替代`, the source must show a clear task replacement, not only a demo.
- If using `炸了`, `逆天`, `成精`, or similar hype terms, pair it with a concrete fact in the same title.
- Avoid empty grand phrases: AI革命, 时代变了, 重塑未来, 颠覆一切.

## Recommended Candidate Mix

For 10 candidates:

- 3 high-impact AI-media versions.
- 2 numeric-hook versions.
- 2 human-impact versions.
- 2 calmer publishable versions.
- 1 search-friendly direct version.
