# Size Reference

Recommended sizes:

| Size | Ratio | Use |
|---|---:|---|
| `1024x1024` | 1:1 | square cover, concept test |
| `1080x1440` | 3:4 | WeChat cover material, Xiaohongshu-style poster |
| `720x1280` | 9:16 | vertical mobile poster draft |
| `1440x1080` | 4:3 | horizontal editorial material |
| `1280x720` | 16:9 | web or video background |

Prompt guidance:

```text
Chinese editorial poster cover, premium magazine composition, clear visual hierarchy, refined lighting
```

For final publishing, keep deterministic text and layout in an upper-layer renderer unless the task intentionally tests Qwen-Image text rendering.
