# 跨平台改写与短视频适配

这份 reference 保留旧版短视频写作和跨平台改写 skill 的可复用规则。

核心原则：平台不是格式差异，而是读者进入内容的方式不同。公众号读者是点进来读，短视频观众是刷到后决定要不要停。改写必须重构，不是复制、压缩或扩写。

## 目录

- 1. 总原则
- 2. 公众号到短视频

## 1. 总原则

跨平台改写前先确认：

- 源内容的核心观点是什么。
- 目标平台的读者或观众是谁。
- 目标平台需要解决什么即时问题。
- 哪些材料必须保留。
- 哪些材料只是源平台里的论证过程，可以删掉或重写。

改写时必须在 YAML 或交付说明里保留 `source`，关联原始素材。

同一主题跨平台时，人设和价值观保持一致，但结构、节奏、标题和结尾要按目标平台重写。

## 2. 公众号到短视频

公众号文章转短视频时，不要把全文压缩成摘要。

动作是：

- 提炼一个最核心的观点，最多一个。
- 删除铺垫、长论证、长背景和复杂分支。
- 保留最有冲突、最有画面、最能让人停下来的材料。
- 强化开头 3 秒钩子。
- 60 秒内讲清一个判断。
- 结尾给观众一个继续说话的位置。

短视频默认结构：

```text
强力钩子 -> 核心冲突 -> 观点输出 -> 例子或动作 -> 结尾留白/互动
```

短视频禁用：

- 引用块格式。
- 长篇论证。
- 超过 30 字的长句。
- 抽象小标题。
- 大段背景介绍。
- 公众号式 intro。

短视频可以使用更短的句子、更直接的口语、更明显的情绪转向，但不能牺牲真实。

## 3. 短视频到公众号

短视频转公众号时，不是把逐字稿加长。

动作是：

- 保留核心观点和最有力的金句。
- 补充论证、数据、案例、来源和边界。
- 把口播里的跳跃逻辑补成读者能读懂的段落。
- 把强钩子改成公众号 intro 和桥接段。
- 用结构标题承载长文阅读节奏。
- 给出更完整的可交付物。

公众号目标不是“更长”，而是“更完整、更稳、更能被复读和收藏”。

短视频里的爽点可以保留，但要放进论证里。不要让公众号变成口播稿的逐字扩写。

## 4. 标题差异

公众号标题：

- 默认接近 30 字。
- 同时承担点击和搜索。
- 要包含对象、痛点、价值、悬念或数字。
- 标题不进入正文层级。

小红书标题：

- 默认约 20 字。
- 可以更情绪化、更生活化。
- 可以适当使用 emoji。
- 不能直接替代公众号发布标题。

抖音和视频号标题：

- 用户主要是刷到内容，标题辅助理解。
- 标题不必承载完整搜索信息。
- 开头 3 秒比标题更重要。

## 5. 教育类短视频风格

旧版短视频风格分析里沉淀过一条教育类内容线。它不是所有 nook 内容的默认风格，但当用户写教育、升学、家庭成长、家长焦虑相关内容时，要保留这些判断。

内容定位：

- 聚焦家长真实焦虑，尤其是中考、高考、择校、班型、政策变化和学习策略。
- 不盲从主流教育神话，允许挑战常见套路。
- 不只追求短期分数和名校叙事，要能把话题拉回自学能力、心理韧性、内驱力、自我认知和长期成长。

表达方式：

- 开头可以直接抛一个家长熟悉的问题、场景或网络说法。
- 语言口语化、生活化，避免教育报告腔。
- 可以有幽默、讽刺和反问，但反问要服务思考，不要变成连续排比。
- 观点要简洁有力，节奏明快。

结构方式：

- 围绕一个具体问题展开。
- 先讲现象，再拆背后的选择成本、政策影响或家庭误区。
- 多用对比和类比，把抽象教育问题讲成家长能理解的日常选择。
- 结尾给出清楚判断，再递一个讨论口给评论区。

作者姿态：

- 像有经验的教育观察者和引导者，不像制造焦虑的营销号。
- 可以直言，但不能恐吓。
- 可以批判，但要给出更清醒的选择方式。

## 6. 输出检查

跨平台交付前检查：

- 是否保留了源内容最核心的判断。
- 是否按目标平台重构，而不是机械压缩或扩写。
- 是否保留 `source`。
- 是否删除了目标平台不适合的格式。
- 是否没有编造新经历、新数据或新案例。
- 是否仍然保留作者自己的判断和人味。
