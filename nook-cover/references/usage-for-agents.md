# Agent 使用说明

当用户要求生成小红书封面时，按下面流程执行。

## 1. 先判断路线

默认路线：

```text
Codex 内置 image2.0 直接出整张封面候选
```

使用外部工具时：

- 快速草图 / 免费测试 / 亚洲人物：`nook-zimage`
- 中文海报 / 中文视觉 / 中文文字测试：`nook-qwen-image`
- 高质量成品 / 图生图 / 参考图保持：`nook-image2-gpt` 或内置 image2.0

## 2. 生成 brief

至少要确定：

- 平台：小红书。
- 标题。
- 主题。
- 视觉系统：人脸冲击封面、手账拼贴种草、产品截图主视觉、近大远小对比封面，四选一。
- 是否有人物。
- 是否允许模型直接生成中文。
- 若中文不准，是否转 HTML / 后处理标题层。

## 3. 输出规则

输出必须进入用户指定目录。没有指定时，进入当前项目的 `output/`。

过程图和候选图不要删除。失败图也可以保留，用于后续复盘。

## 4. QA

必须逐张检查：

- 中文是否正确。
- 标题是否足够大。
- 人物是不是完整半身 / 三分之二身，而不是头像。
- 标题有没有遮脸。
- 撕纸、贴纸是否自然。
- 是否像小红书，而不是 PPT。

如果失败，不要只改颜色。要重新选择构图机制或提示词。
