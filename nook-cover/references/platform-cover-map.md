# 1 平台封面差异表

本文件记录 `nook-cover` 的平台差异。它不是完整规范，只负责帮助 Agent 判断不同平台封面的设计侧重点。

## 1.1 核心原则

使用同一套封面生产主流程，再按平台 profile 分支。

共享流程：

```text
内容压缩 → 标题钩子 → 视觉层级 → 平台安全区 → 最终渲染
```

## 1.2 平台 Profile

| 平台 | 封面特点 | 设计重点 |
|---|---|---|
| 小红书 | 竖向或方形信息流首图 | 强标题、审美感、可读性、收藏 / 点击冲动 |
| 微信公众号 | 横向文章头图 | 编辑感、品牌一致性、文章气质 |
| B站 | 16:9 视频封面 | 点击欲、标题可读、人物 / 物件 / 主题证据 |
| YouTube | 16:9 视频缩略图 | 大主体、少文字、强对比、好奇心 |
| 抖音 | 竖版视频封面 | 中心安全区、短标题、视频原生场景 |
| 视频号 | 竖版或信息流封面 | 移动端标题安全区、账号一致性、平台 UI 遮挡意识 |

## 1.3 V0.1 聚焦

先只实现小红书封面。

在一个平台能稳定产出可用结果之前，不扩展多平台输出。
